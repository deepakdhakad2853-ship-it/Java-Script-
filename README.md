1️⃣ What is a variable in JavaScript?

“Variable JavaScript में एक storage container होता है जिसमें हम किसी value को store करते हैं। इसका उपयोग data को hold, update और process करने के लिए होता है।”

4️⃣ What is variable hoisting?

“Hoisting एक JavaScript mechanism है जिसमें variable declarations को execution से पहले top पर उठा दिया जाता है, लेकिन var को undefined value मिलती है जबकि let और const hoist तो होते हैं पर initialize नहीं होते।”

6️⃣ What is the scope of variables in JavaScript?
✔️ Answer:

JavaScript में main 3 प्रकार के scope होते हैं:

Global scope → पूरे program में available

Function scope → केवल उसी function के अंदर

Block scope → {} braces के अंदर (let और const)

7️⃣ Can we update const variable?
✔️ Answer:

Primitive value नहीं बदल सकते

लेकिन object या array की internal values बदल सकते हैं
